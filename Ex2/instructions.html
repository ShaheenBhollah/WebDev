Client-Side Web Development CA3

Complete parts 1, 2 and 3 as separate assignments. You may work on them in any order but keep in mind that the part numbers also denote the level of difficulty (1 being the easiest and 3 the most difficult).

Part 1 [40%]

  1. Open the file part1.html in a text editor and in a browser.
  2. Create a new file called part1.js and open it in a text editor.
  3. Include the Javascript file in the HTML file, by adding a script element as the last element inside the body element.
  4. In the Javascript file, declare and define a function called writeGreeting with no parameters. The function should write the following text into the element with id 'info':

    Hello, my name is <your name>.

    where <your name> is a placeholder. So, if your name is Jimmy Jones, then you will be writing the sentence

    Hello, my name is Jimmy Jones.

  5. In the HTML file, set an event handler on the button friendly so that when that button is clicked, the function writeGreeting is executed.
  6. In the Javascript file declare and define a function called modifyBorderColor with one parameter called element_id. The function should set the style property borderColor to value 'green' on the element whose id is contained in element_id.
  7. In the HTML file, add the id 'to-style' to the button Web.
  8. In the HTML file, set an event handler on the button I so that when the mouse goes over that button (event 'mouseover'), the function modifyBorderColor is called with argument 'to-style', in order to modify button Web.
  9. Reload the HTML page in the browser and perform a final test to see that:
    • when the button friendly is clicked, the text Hello my name is <your name>. appears at the bottom of the page
    • when the mouse pointer goes over button I, the button Web has its border colour changed

Part 2 [35%]

  1. Open the file part2.html in a browser. Open the file in a text editor, but for viewing only, as you should not modify the HTML file in this part.
  2. The file part2.js is included in the HTML as an external script. This Javascript file is the only file you should edit in this part.
  3. The file part2.js contains definitions for the words am, your and page, stored in a Javascript object.
  4. Declare and define a function called processWord with a single parameter called word and the following functionality:
    • if one of the three defined words (am, your or page) is passed in, the function fetches the word's definition from the provided object and stores it in a variable called info
    • if any other string is passed in, an empty string is stored in variable info
    • the value from the HTML input element is read into a variable called inputVal and
      • if the input is 'upper', the value in info is transformed to all-uppercase
      • if the input is 'space', each space character in info is replaced (use string method replaceAll) with three space characters i.e. '&nbsp;&nbsp;&nbsp;'
      • if the input is any other string, the value in info is not changed
    • write the value of variable info into the element with id 'info'
  5. Write some code outside of any function i.e. directly in the Javascript file to do the following:
    • fetch the objects corresponding to the three elements with words am, your and page (use method getElementsByClassName)
    • set the colour of these three elements to 'darkgray'
    • set handlers on the elements (use method addEventListener), so that when any of these elements is clicked the function processWord is called with the clicked element's content as the argument
  6. Reload the HTML page in the browser and perform a final test to see that the three elements with words am, your and page become darkgray as soon as the page is loaded and that clicking on any of them brings up the definition, formatted according to the additional instruction ('upper' or 'space') in the input field.

Part 3 [25%]

  1. Open the file part3.html in a browser. Open the file in a text editor, but for viewing only, as you should not modify the HTML file in this part.
  2. The file part3.js is included in the HTML as an external script. This Javascript file is the only file you should edit in this part. The Javascript file contains some code that you can use:
    • a function, get_shuffled_indices(n), which returns an array of numbers 0 to n-1, shuffled
    • an array with the element labels
    • an array of values (see the problem description)
  3. Write Javascript code that does the following:
    • When the HTML page is loaded, the borderWidth style property is set on the seven visible elements to the following values, one value per element in random order:

      '1px', '3px', '5px', '7px', '9px', '11px', '13px'

    • As the user clicks on the elements, if an element is clicked out of order the following message appears instead of the instruction line: "You have made a mistake. Reload the page to play again."
    • If the user clicks on all seven elements in the right order, the following message appears instead of the instruction line: "Well done! Reload the page to play again."
